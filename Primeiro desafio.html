HISTÓRIA DA COMPUTAÇÃO

(Voltar)

O primeiro computador

Os computadores de primeira geração surgiram a partir de 1951, que funionavam por meio de válvulas eletrônicas e circuitos. Neste ano foi lançado o primeiro computador, o ENIAC que foi desenvolvido a pedido do exército dos Estados Unidos e possuiam acesso restrito e de grande consumo de energia.


Sugirmento da Internet

(Voltar)

Tudo se iniciou devido à Guerra Fria que acontecia em 1957 entre os Estados Unidos e a União Soviética. Os estados Unidos em prol de sua proteção, queriam uma maneira de eficiente de proteger suas informações e comunicações a um possível ataque nuclear soviético. Devido a esses esforços e a inovação fui iniciado a tragetória do que chamamos hoje de Internet.

Em 1957 a União Soviética lançou o primeiro satélite o Sputnik, e os Estados Unidos pensaram que o motivo seria para espionagem. O presidente militar estadunisense da época fundou uma agência voltada especialmente para estudos tecnológicos para guerra a DARPA, que em 1969 a tecnologia principal que criaram foi a Arpanet que resolveria a preocupação de um possível ataque nuclear da União Soviética.

A Arpanet inicialmente era uma pequena rede de quatro pontos ligados por computadores: um SDS SIGMA 7 na Universidade da Califórnia, um SDS 90 na Universidade de Stanford, um IBM 37075 no Centro de Matemática de Los Angeles e um DEC PDP-10 na base militar de Utah.

A partir desta rede, verificaram que os computadores não se comunicavam muito bem já que tinham linguens diferentes pois eram de fabricantes diferentes. Então alguns pesquisadores da Universidade da California criaram o Protocolo NCP, que tinha uma linguagem única para conseguir a comunicação entre os computadores

Posteriormente foi criado dois protocolos que se unificaram: o TCP que resolvia o problema do NPC de parar a rede para fazer uma nova conexão, mas que teria que ser aprimorado já que não identificava os computadores devido a quantidade na rede, assim surgiu o IP (Internet Protocol) criados, respectivamente, pelos pesquisadores Robert Kahn e Vint Cerf. A junção dos dois trabalhos resultou no protocolo TCP/IP que utilizamos até hoje com um versão mais avançada.

O que são Clients e Servers

(Voltar)
Clients

Client é tudo aquilo que consome produtos ou serviços de algum lugar, na tecnologia Client são aqueles que consomem a informação no caso os aparelhos (celulares, computadores, notebooks).

Servers

Como no nome já explicativo, Server são aqueles que servem as informações para os Clients.

Comunicação do Client-Server

A comunicação é feita pela Internet, por exemplo um notebook (client) conecta ao Servidor (Server) através da Internet, recebendo as informações que o servidor processa retornando arquivos estáticos como: HTML, CSS, JavaScript, imagens, etc...

ESTRUTURA DO HTML

(Voltar)

A estrutura básica do HTML é composto pela seguinte forma:

  • DOCTYPE informa qual o tipo do arquivo a ser criado;
  • Html informa a página em que o código será inserido;
  • Head é tudo aquilo que carrega antes de aparecer ao usuário;
  • Body é o corpo do site, ou seja, tudo aquilo que o usuário vê na página.

Atributos básicos das Tags

(Voltar)

São as propriedades que uma Tag pode ter: as globais (todas as tags vão ter) ou as específicas de cada Tag.

Genéricas

  • ID que é um identificador do elemento, usada para formatações futuras com o CSS, por exemplo;
  • Style com o Strong (negrito) que permite inserir comandos do CSS direto no elemento. Ex: style="color: blue", deixa seu elemento com a fonte azul e negrito;
  • Class também utilizado para formatação mais voltado para padronização. Ou seja, ele formata em grupos. Ex: Class="título" todos elementos nomeados com "título" terão a mesma formatação.

Específicas

  • Input insere um campo com bordas texto no HTML e ele tem um atributo "type" que faz com que você escolha qual deve ser o tipo de preenchimento daquele campo ex: enviar, check, números, cores...
  • Img permita que insira uma imagem e para a imagem aparecer é preciso do atributo "scr" onde você insere o link da imagem ou o caminho da pasta onde ela esta. Ex: img scr ="link" ou img scr ="/Imagens/teste_Colheita.jpg".

Tags para textos

(Voltar)

Aqui iremos explicar as principais tags utilizadas para inserir textos em sua página: títulos, parágrafos e formatações.


Títulos

As tags utilizadas para titulos são h1, h2, h3..., h6 e seus tamanhos possuem diferenças seguindo uma ordem Decrescente:

h2 Título

h3 Título

h4 Título

h5 Título
h6 Título


Parágrafos

Todo texto deve ser utilizado e a cada novo parágrafo deve ser inserido uma nova tag "p"

p Texto que aparece ao usuário p


Formatações de Texto

  • Citação: podemos inserir uma citação de outro autor com a tag blockquote que insere uma margem antes do texto, por exemplo:
    "Gosto do impossível porque lá a concorrência é menor." - Walt Disney
  • Negrito: utilizamos a tag "strong" para deixar as palavras em negrito;
  • Itálico: utilizamos a tag "i" para deixar as palavras em itálico (deitadas);
  • Sublinhado: utilizamos a tag "u" de undeline para deixar uma palavra sublinhada;
  • Grifado: utilizamos a tag "mark" para deixar palavras grifadas com aspecto de marca texto;
  • Sobrescrito: utilizamos a tag "sup" para deixar algo sobrescrito aqui;

Listas

(Voltar)

Existem tipos de Listas as Ordenadas, Não ordenadas e Descrições de lista. Aqui vamos explicar melhor sobre cada uma.

Listas Ordenadas

É o tipo de lista como já diz o nome com ordem e pode ser utilizada com os sistemas de número indo-arábico, Romano e por letras minúsculas e maiúsculas.

Utilizamos a tag "ol" de ordered lists para abrir o campo das linhas e "li" de line para inserir cada tópico e para alterar o sistema de listagem, utilizamos o "type" após a tag para aterar o tipo:

  1. Type="1" - Números indo-arábico
  1. Type="I" - Números romanos maiúsculo
  1. Type="i" - Números romanos mainúsculo
  1. Type="A" - Letras maiúsculas
  1. Type="a" - Letras minúsculas

Listas Não-ordenadas

Ao contrário das listas ordenadas, como já descreve o nome este tipo de lista não tem ordem, apenas os tópicos.

Agora utilizamos a tag "ul" de "Unordered Lists" para abrir e "li" para as linhas e podemos alterar com o "type" qual o tipo dos tópicos. Sendo os mais conhecidos por círculo, bolinha, quadrado ou nenhum.

  • Type="disc" - Disco/Bolinha (círculo preenchido)
  • Type="circle" - Círculo (Sem preenchimento)
  • Type="square" - Quadrado
  • Type="none" - Nenhum

Descrição de lista

Este tipo de lista gera uma listagem e logo abaixo do item, insere um margem para uma descrição.

Agora utilizamos a tag "dl" de "Description Lists" para abrir o campo da listagem, "dt" "discription term" para os itens e "dd" para a descrição dos itens

"dt" - Café
"dd" - preto e forte
Leite
Integral ou desnatado

Links

(Voltar)

Os links são utilizados para a navegação entre páginas e para adicionarmos utilizamos a tag "a" de "ancor" para ancorar à algum lugar. A formatação do link é padrão então a frase ficará sublinhada e de cor difernete, mas com o CSS podemos alterar depois.

Mas não utilizamos apenas a tag "a", precisamos de um atributo para inserir qual a ligação a ser feita, por isso, utilizamos o "href" que é a referência dessa âncora e com ele adicionamos nossos links.

Site da DIO.me

Temos outro atributo que podemos utilizar dentro da tag "a" que é o "taget". Nele podemos utlizar as opções de como o link vai se comportar, como por exemplo, abrir na mesma aba ou em outra.

  • Target="_self" - abre na mesma aba (comportamento padrão da tag "a")
    • Site da DIO.me
  • Target="_blank" - abre em outra aba
    • Site da DIO.me

Podemos adicionar também outro atributo na tag "a", que se chama "title". Ele cria um balão que ao passar por cima do link, um tíyulo apareça.
